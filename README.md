# Interactive Dice & Station Selector

This repository contains two small JavaScript projects:

1. **Interactive Dice**: A simple animation that simulates rolling a dice.
2. **Station Selector**: An interactive application to explore train stations by line.

---

## Features

### Interactive Dice
- Simulates dice rolls with smooth animation.
- Displays the result of the roll after each animation.

### Station Selector
- Displays a list of train lines and their stations.
- Dynamically updates station boards based on user interaction.
- Navigate stations using buttons or keyboard arrows.
- Loads station data from a JSON file.
